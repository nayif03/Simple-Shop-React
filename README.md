# A React Shopping Cart Demo

This is a small demo to show how the State can be used in React (**without Redux**) to create a shopping cart.


## Techs
- React  
*Components, Props & States, Lifecycle methods*
- React Router  
*Simple routes and NotFound*
- Bootstrap
- FontAwesome
